Student Management System
Built a basic student management system with Python and MySQL for managing student records, grades, and attendance. Implemented CRUD operations and a simple CLI interface.
Tech: Python, MySQL
Description:

A command-line application to manage student data, including adding, updating, deleting, and viewing records. Data is stored in a MySQL database.

  • Add, update, delete, and view student records
  • Simple command-line interface
  • Uses MySQL for persistent storage
Full Code Example (Python): 
import mysql.connector

def connect():
    return mysql.connector.connect(
        host="localhost",
        user="root",
        password="yourpassword",
        database="students"
    )

def show_students():
    conn = connect()
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM student_records")
    for row in cursor.fetchall():
        print(row)
    conn.close()

def add_student(name, grade):
    conn = connect()
    cursor = conn.cursor()
    cursor.execute("INSERT INTO student_records (name, grade) VALUES (%s, %s)", (name, grade))
    conn.commit()
    conn.close()

# Example usage:
add_student("John Doe", "A")
show_students()

Basic Calculator (C Language)
Created a command-line calculator in C that performs basic arithmetic operations and handles user input errors gracefully.
Tech: C
Description:

A simple calculator program written in C that supports addition, subtraction, multiplication, and division.

  • Handles basic arithmetic operations
  • Command-line interface
  • Input validation for operators
Full Code Example (C): 
#include <stdio.h>
int main() {
    char op;
    double a, b;
    printf("Enter operator (+, -, *, /): ");
    scanf(" %c", &op);
    printf("Enter two numbers: ");
    scanf("%lf %lf", &a, &b);
    switch(op) {
        case '+': printf("%.2lf\n", a+b); break;
        case '-': printf("%.2lf\n", a-b); break;
        case '*': printf("%.2lf\n", a*b); break;
        case '/': printf("%.2lf\n", a/b); break;
        default: printf("Invalid operator\n");
    }
    return 0;
}
